# 📊 Postepay_Dashboard
Una dashboard interattiva locale costruita in Python (Dash/Plotly) per il tracciamento, la categorizzazione automatica e la previsione delle spese bancarie (es. Postepay, conti correnti).

## 🚀 Come avviare l'app

1. Clona questo repository.
2. Crea un ambiente virtuale (opzionale ma consigliato): `python -m venv venv`
3. Installa le dipendenze: `pip install -r requirements.txt`
4. Avvia l'app: `python app.py`
5. Apri il browser all'indirizzo `http://127.0.0.1:8050`

## 🔒 Privacy First
L'app gira interamente in locale sul tuo PC. I tuoi dati finanziari non lasciano mai la tua macchina. I file Excel vengono salvati nella cartella `data/` con un sistema di backup automatico a ogni modifica.
